Summary

Build instructions for an open source device that detects one radio-frequency-identification-tag implanted mouse and passes it through to another area. Useful for self-paced behavioral tasks in neuroscience, so animals can be co-housed and automatically allowed to enter an area designed for measuring only one animal. Off-the-shelf hardware from Sparkfun and Arduino and Python programming are used in this device.
