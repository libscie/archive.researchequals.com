Summary

This module provides a list of URLs to publicly available datasets on the OSF. These URLs are collected from the public API and can be used to easily download datasets that have previously been shared. The module provides links to 69,214 open datasets hosted on the OSF, in the timeframe 2022-01-01 through 2022-06-31. These URLs can be used to directly download the datasets - please note that some links may break in the future and that if you use this you will need to account for no longer available files. In the supporting files, a script is provided that can be adjusted to retrieve datasets in other date ranges (requires NodeJS >19.0.0).
