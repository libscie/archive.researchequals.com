Summary

In this module we present a figure on the growth of journal articles. It shows the number of articles available through CrossRef by year of publication. This figure is on a log scale, where a straight line would indicate exponential growth. The data is included in the supporting files, along with the shell script used to collect the data and the R script to create the plot.
